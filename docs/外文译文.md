## 摘要

随着具身智能代理和室内服务机器人的兴起，物体检测已经成为支持语义建图、路径规划和人机交互的关键组件。然而，室内场景通常面临着如严重遮挡、巨大的尺度变化、小且密集的物体分布以及复杂的纹理等挑战，这使得现有方法在鲁棒性和准确性方面均陷入困境。本文提出了一种基于 Hölder 正则化引导的多域融合检测框架 MDF-YOLO。在主干网络、颈部网络和特征恢复阶段，该框架引入了正交网格记忆模块 (CrossGrid Memory Block)、基于 Hölder 正则化引导的层次上下文聚合模块 (HG-HCA)，以及频率引导的残差模块 (FGRB)，实现了跨状态空间、空间域和频率域的互补特征建模。特别地，HG-HCA 模块利用 Hölder 正则化图作为引导信号，来平衡宏观和微观路径之间的动态均衡，从而实现了全局一致性和局部判别力的自适应协调。实验结果表明，MDF-YOLO 在 mAP@0.5、mAP@0.75 和 mAP@0.5:0.95 等指标上显著优于主流检测器，分别达到了 0.7158、0.6117 和 0.5814，同时在 FPS 和延迟方面保持了接近实时的推理效率。消融研究进一步验证了 CGMB、HG-HCA 和 FGRB 在改善小目标检测、遮挡处理和跨尺度鲁棒性方面的独立与协同贡献。本研究展示了 Hölder 正则化与多域融合建模在物体检测中的巨大潜力，为复杂室内环境下高效视觉建模提供了新思路。

**关键字:** 室内物体检测；Hölder 正则化；多域融合；上下文聚合；频域细节恢复

## 1. 引言

随着人工智能和机器人技术的快速进步，具备自主感知周围环境、制定决策及执行动作能力的智能系统——即具身智能代理，正日益融入日常生活中[1,2]。在室内环境中，服务机器人和清洁机器人等设备被广泛部署，它们严重依赖于高度准确的视觉感知来支持自主导航、人机交互以及场景理解等任务[3,4]。在这一框架内，物体检测扮演着核心角色，通过识别和定位物体，为包含语义地图构建[5,6]、路径规划[7,8]以及任务级决策[9]等下游模块提供关键的空间与语义信息。因此，开发兼具高精度与高鲁棒性的室内物体检测算法，对于提高具身智能代理的整体可靠性和有效性至关重要。

然而，室内环境固有的复杂性对物体检测构成了实质性的挑战[10]。首先，物体往往遭受严重的遮挡与混乱堆放，导致其特征碎片化，这需要模型具备强大的全局推理能力[11]。其次，从大型家具到小型桌面物品的巨大尺度变化，要求有效的多尺度表示与融合[12]。第三，小目标的识别严重依赖于如边缘和纹理等高频细节，而这些细节在深度卷积网络的下采样过程中经常会退化或丢失，导致漏检或误检[13]。因此，同时解决全局上下文推理、尺度适应性和精细细节保留，依然是室内物体检测的核心难点。

为了应对通用目标检测的挑战，在过去十年中，以卷积神经网络 (CNN) 为中心的检测器经历了持续演进[14]。其中，单阶段检测器凭借精简高效的设计，在实时应用中占据了主导地位。它们的研究和发展轨迹始终致力于寻求准确性与效率之间的平衡。这一演变涵盖了将检测统一为回归问题的 YOLO [15]、引入 Focal Loss 以有效解决正负样本极度不平衡的 RetinaNet [16]，以及像 FCOS 这种无锚框 (Anchor-free) 方法的兴起[17]。单阶段检测器的性能与范式在不断地创新。与此同时，在移动端和边缘计算需求的驱动下，诸如 MobileNet 和 ShuffleNet 等轻量级主干网络得到了广泛的研究和应用[18]，催生了像 EfficientDet 这样在资源受限平台上表现出色的检测器[19]。尽管 CNN 在物体检测上取得了广泛成功，但其本质上仍受限于局部的感受野，引入了局部归纳偏置[20]。这意味着 CNN 在捕捉如边缘和纹理等细粒度局部特征时极为有效，但在对图像中相距较远区域之间的长程依赖性建模时却显得吃力（例如，当桌子的表面和桌腿相距甚远时识别它们之间的关联）。虽然这种偏置提高了学习局部模式的效率，但却阻碍了基于 CNN 的检测器在复杂室内场景中进行整体推理的能力，从而降低了它们对严重遮挡和尺度变化的鲁棒性。

为了克服 CNN 的局部性瓶颈，以 ViT 为代表的视觉 Transformer 架构应运而生。其核心的自注意力机制在捕捉全局上下文方面极为有效。在此基础上，DETR 框架将目标检测重构成了一个端到端的集合预测问题[21]。然而，Transformer 强大的全局建模能力伴随着巨大的计算代价。其自注意力机制的复杂度随输入分辨率呈平方级增长，导致高分辨率下训练和推理成本高昂直至不可接受[22]，使得难以满足室内应用严苛的实时性要求。

针对上述挑战，当前的研究轨迹主要包含以下几个关键方向。一方面，研究人员致力于探索更高效的长程依赖建模机制。这涵盖了从各种新颖的注意力模块到近期在视觉领域备受瞩目的状态空间模型 (SSMs) [23,24]，它们均旨在保持线性计算复杂度的同时实现全局上下文感知。另一方面，学术界正在研究更加精密的多尺度特征交互与融合模块。这包括通过注意力机制强化关键特征表达[25]，或者结合如 3D 点云那么多模态信息，帮助系统理解复杂的空间关系[26]。此外，从如频域等新角度切入，通过明确补偿网络传播过程中丢失的细节以恢复特征，也已成为提高小目标检测性能的有效策略[27]。

基于前述分析，本文提出了 MDF-YOLO，一款专门针对室内场景量身定制的多域融合物体检测器。我们的目标是通过一种协同优化的解决方案系统地解决室内检测的核心难题。该方案在全局上下文建模、多尺度融合以及细节恢复方面取得了性能突破，同时保持了实时效率。本文主要贡献如下：
1. 我们设计了一种名为正交网格记忆模块 (CrossGrid Memory Block, CGMB) 的新型主干网络模块。该模块并行整合了状态空间建模与局部卷积路径，通过正交网格记忆机制以线性复杂度捕获全局上下文和长程空间依赖关系，从而有效解决严重遮挡的问题。
2. 我们提出了基于 Hölder 正则化引导的层次上下文聚合 (HG-HCA) 模块。该模块在基于 Hölder 的正则化先验的引导下，整合了宏观和微观上下文路径。在此计算出一个可微的正则化图来表征局部的平滑度和粗糙度。通过一个轻量级的校准机制，该正则化图被转换为面向任务的逐像素引导信号。这一设计使得网络能够动态地平衡全局结构一致性和局部判别细节，从而提高对室内场景中尺度突变、复杂纹理和密集遮挡的鲁棒性。
3. 我们引入了一个名为频率引导残差模块 (FGRB) 的上采样模块。该模块通过并行的频域补偿路径扩展了空间上采样路径。它利用可学习的频率权重矩阵增强高频分量，并以残差方式恢复图像细节，从而提高了小目标的检测性能。

在公开室内场景数据集上的大量实验结果表明，MDF-YOLO 在准确性和效率上均超越了现有的主流方法，验证了所提出的基于多域融合的物体检测网络的有效性和新颖性。

## 2. 相关工作

### 2.1 传统的基于规则与手工特征的方法
在深度学习广泛普及之前，室内物体检测的研究主要具有两方面的特征：由传感器驱动的基于几何规则的方法，以及基于手工提取特征的计算机视觉方法。基于传感器的方法利用从超声波、红外线和激光测距仪获取的几何信息，通过局部避障和路径校正来确保运动安全。通过持续的测距和沿墙策略，实现了低计算约束下的稳定避障；以及借助基于栅格贝叶斯机制建立障碍网格图。但这部分由于缺乏分类属性特征而在此不详细展开。

随着图像传感器和计算平台的成熟，研究焦点转移至由手工设计的视觉特征和浅层判别分类器组成的检测范式。Lowe 的 SIFT 在尺度空间进行关键点检测，并利用方向对齐的局部梯度直方图表示外观，展示了对视角和光照变化的鲁棒性。Bay 等人的 SURF 利用积分图和盒式滤波器近似 Hessian 响应，大幅降低了特征描述子的计算成本；而传统最鼎盛的 HOG 加 SVM 技术架构甚至使得此类方案成为了早期滑窗监测的配置标准。随后的可变形部件模型 (DPM) 更是进一步优化了特征匹配技术。但总的而言仍然受制于室内跨域遮挡因素的强烈影响。伴随 RGB-D 设备的普及，使用其 3D 几何特性或是采用点云检测（例如 FPFH）确实解决了一部分问题，但终究受制于复杂的调参设计以及计算效率的困扰。

### 2.2 通用深度学习模型
深度学习促成了物体检测从手工特征和浅层分类器向端到端可学习表示方向转变的范式更迭。R-CNN 将候选区域生成与 CNN 特征学习相结合，在通用数据集上显著提升了检测准度，确立了以深度特征为中心的检测基准。紧随其后的如 SPP-Net, Fast R-CNN 及最为盛行的 Faster R-CNN，统合了区域生成机制和卷积流将检测带向极佳统一方案阶段；至于单阶段的诸如 SSD 等及整个的 YOLO 宇宙，更是因抛却冗杂机制成为工程上兼备极高实时执行能力的标杆。尤其是当下的 YOLO 系列加上 Focal Loss，配合各式各样由如 MobileNet 启发的轻型骨干算力支持下，即使遭遇极端且比例反差的数据一样可以交出不俗答案。但在面对室内情况，因为常规主干受到固有的 CNN 定向区域影响依然限制了在极端大背景遮挡推理上的泛用，且容易使得各种特征细节点被直接降维过滤流失。

### 2.3 针对室内场景挑战的探索
为修补通用架构在此的盲区，业界开展了三大主要针对工作方向：全局上下文建模、跨尺度语义对齐及细节复原。在针对遮挡拥挤等情况下开展全局架构模型关系推理往往面临资源成倍增加或是运算图开销难以约束的问题（诸如关系网络 Relation Networks，Occlusion-aware R-CNN 等等）；跨尺度对齐由于包含大量多态融合 (如 ASFF 及 NAS-FPN 模型) 确乎具备提升鲁棒性适应极高尺度拉跨度问题效用，在实施的稳定控制却很难标准化。细节与可见性的频域级补偿或采用 CARAFE 空间注意力提升的方法确实可以拉升纹理识别但难以处理额外参数。针对此部分引入可明确反映且微分引导的高正则连续特征不仅能在成本端减去额外负担并且完全能够补齐在光度，复杂密集干扰时的局部引导判定基准（Regularity prior）。

## 3. 方法论

### 3.1 MDF-YOLO 网络架构
室内目标检测任务，特别是在严重遮挡、极大尺度变化和密集小目标分布的情况下，对检测器的多层级特征建模能力提出了极高的要求。近年来，YOLO 系列由于其单阶段架构等优势已成业界标配，其中 YOLOv8 具有主干、颈部及检测端多维解耦机制从而成为理想蓝本。由于 YOLOv8 在深层长距离卷积受限以及细节容易流失的问题，本论文因此提出多域融合的 MDF-YOLO：系统架构图中（参见原文 Figure 1）依次于对应区域整合植入了：拥有正交长测建模能力的主干 CGMB 模块、用于平衡及通过 Hölder 正则指引完成跨尺度局部与结构化交互信息的 HG-HCA 颈位系统、以及处于最后高解析重构建阶段起引导恢复重构高频边缘分频细节的 FGRB 组件；使在涵盖不同解析阶面上都能协同处理，打破孤立算法下的局限，同时确保了检测效率。

### 3.2 跨网格图记忆模块 (CrossGrid Memory Block, CGMB)
CGMB 的核心原则基于通过正交网格记忆建模 (OGMM) 显式地维持 2D 拓扑结构。该机制沿水平和垂直相互正交的方向将输入特征向量化，并通过连续增加具备位置加强指引余弦属性序列，借助于 Mamba 状态空间法执行空间跨度的远距连接特征截取记忆，同时伴随残余常规 2D 本地细部流完成最后统合。以此克服普通直接全维展平带来的彻底结构破坏错误，进而完全保证特征不会因遮挡发生彻底截断或丢失对应连通判定。
数学公式与流程表示为：对于局部细化输出： $F_{local} = C2f(F_{in})$
对于正交多维长距分析： $F_{global} = Unflatten[Mamba_h(S_h + PE_h)] + Unflatten[Mamba_v(S_v + PE_v)]$
最终汇总为结合本地细化加上残差合并映射的组合结果： $F_{out} = Conv(F_{in} + F_{local} + F_{global})$。

### 3.3 Hölder 正则化引导层次上下文聚合模块 (HG-HCA Block)
由于极大范围尺寸区别以及复杂的被挡对象，模块的核心要领完全受控并基于预先处理完毕的 Hölder 正则空间先验参数信号（HP-head）。
给予一个特征原图，HP-head 通过计算并模拟近似小波域衰减率给出代表这局片区域内其纹理几何强硬或粗糙平滑度的图相矩阵 $\tilde{\alpha}(x)$。该部分会因为不同粗糙细节产生极有指导性的偏差并由此经过 Sigmoid 收束出对应指示信息：$\alpha^*(x) = \sigma(\gamma(\tilde{\alpha}(x) - \tau))$。
而对应给出的两分量独立权值 $(w_{macro}(x)$ 与 $w_{micro}(x))$ 就起到了排他性引导效力，这保证了全局上下文特征重组计算以及极微观细节补充模块在面对高平滑或剧变杂碎波段下可以精准使用适用的重整比例系数！

对于微观宏观补充路，其计算在分别获得 $F_{macro}$ 和对应细节组 $F_{micro}$ 的极有效特征补偿后实现了完全平稳的双向适应。
